# 🌦️ Weather Forecast & Air Quality Dashboard

## 📌 Short Description

An interactive and visually engaging Power BI dashboard built to analyze real-time weather conditions, temperature forecasts, air quality metrics, and environmental indicators across multiple cities. The dashboard provides actionable insights into weather trends, pollution levels, rainfall probability, and atmospheric conditions to support better environmental awareness and decision-making.

## 🛠 Tech Stack

* **Power BI Desktop** – Main platform used for creating interactive reports and visualizations
* **Power Query** – Data cleaning, transformation, and preprocessing
* **DAX (Data Analysis Expressions)** – Used for calculated measures, AQI indicators, and KPIs
* **Weather API Dataset** – Real-time weather and air quality data source
* **Excel / CSV** – Data preprocessing and formatting
* **Custom UI Design** – Modern dark-themed dashboard styling
* **File Formats:** `.pbix` for development and `.png` for dashboard previews

---

# 📂 Data Source

**Source:** Weather & Air Quality API Dataset

### **Dataset Includes:**

* Temperature and weather conditions
* Humidity and wind speed
* Visibility and atmospheric pressure
* UV index and precipitation
* Sunrise and sunset timings
* Air Quality Index (AQI)
* Pollution metrics (PM10, PM2.5, CO, NO2, SO2, O3)
* Weekly weather forecast
* Rainfall probability analysis

The dataset helps analyze environmental conditions, weather forecasting patterns, and air pollution levels across different cities.
